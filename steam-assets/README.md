# Steam Artwork

Replace the placeholder files here with your own artwork, then apply them in Steam.

## Files and sizes

| File | Size | Used for |
|---|---|---|
| `cover.png` | 600×900 | Portrait grid cover (main library view) |
| `hero.png` | 1920×620 | Wide banner at top of game detail page |
| `capsule.png` | 460×215 | Horizontal capsule (some library views) |
| `icon.png` | 256×256 | Small icon shown next to game name |

## How to apply in Steam

1. Right-click **Phonic** in your Steam library
2. Click **Manage → Set custom artwork**
3. Select the file for that slot
4. Repeat for each image type

## Automated option — SteamGridDB Manager

If you upload your artwork to [SteamGridDB](https://www.steamgriddb.com), the
[SteamGridDB Manager](https://www.steamgriddb.com/manager) app can apply it
automatically across all your non-Steam games in one go.

## Notes

- PNG is preferred. Steam also accepts JPEG for most slots.
- Keep a transparent background on the logo/icon if you want it to composite cleanly over the hero.
- Steam may cache old artwork — restart Steam if changes don't appear.
